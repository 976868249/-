## 设备及软件要求
Windows 10及以上、谷歌浏览器、指纹浏览器
## 功能明细
**一、自热模块**

- 采集笔记文案（标题、内容、标签、图片or视频）
- 批量管理小红书账号，矩阵运营
- 自动发布制作好的笔记
- 支持创建文案库，随机抽取文案发布

**二、RPA采集模块**


- 支持关键词筛选，自动化采集符合条件文章。
- 支持评论采集，根据关键词采集精准用户
- 支持筛选评论时间及关键
  
## 使用方法
 1. 下载exe文件
 2. 注册并登录软件
 3. 设置好采集关键词，先采集文章链接
 4. 用文章链接采集指定关键词
 
**注意：需要以管理员身份运行本软件**
